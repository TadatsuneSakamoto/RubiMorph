# Security And Privacy

## 方針

RubiMorph は Windows ローカル実行を中心にした変換ツールです。現在の公開版では原稿本文を外部サーバーへ送信しません。

## 監査項目

| 項目 | 状態 |
| --- | --- |
| 原稿本文を外部送信しない | 対応。ネットワーク通信処理なし。 |
| 外部API前提になっていない | 対応。標準ライブラリ中心。 |
| 入力ファイルを勝手に上書きしない | 対応。出力先が同じ場合 `_converted` を付ける。 |
| 原稿フォルダを勝手にスキャンしない | 対応。ユーザー指定フォルダのみ処理。 |
| 認証情報を保存しない | 対応。認証機能なし。 |
| ログに本文全文を保存しない | 対応。GUI/CLI はファイル名、警告、件数中心。 |
| エラー時に本文全文を大量表示しない | 対応。例外メッセージ中心。 |
| 変換サンプル以外の本文を混入させない | 対応。tests/golden は短いサンプルのみ。 |

## 残課題

- 配布版 exe の署名方針。
- ログファイル出力機能を追加する場合の本文扱い。
- Web 版を追加する場合の通信・保存設計。
