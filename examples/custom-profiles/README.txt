RubiMorph custom profile samples

Files
- example-bracket-format.rubimorph-profile.json
  Sample custom profile. It converts ruby to [[ruby:base|reading]] and emphasis to [[em:text]].
- example-input.txt
  Small Kakuyomu-format input for checking the sample profile.
- example-expected-output.txt
  Expected output when example-input.txt is converted from Kakuyomu to Example Bracket Format.
- author-work-sample-kakuyomu.txt
  Kakuyomu-format author text sample.
- author-work-sample-custom-format.txt
  Output generated by RubiMorph from author-work-sample-kakuyomu.txt to Example Bracket Format.
- author-work-sample-roundtrip-kakuyomu.txt
  Output generated by RubiMorph from author-work-sample-custom-format.txt back to Kakuyomu.

Import
1. Open RubiMorph.
2. Open the Custom Format menu.
3. Open the custom format manager.
4. Import example-bracket-format.rubimorph-profile.json.
5. Validate it, then enable it.

Use
1. Select Kakuyomu as the source.
2. Select Custom: Example Bracket Format as the target.
3. Convert example-input.txt or author-work-sample-kakuyomu.txt.
4. Compare the result with example-expected-output.txt or author-work-sample-custom-format.txt.
5. For reverse conversion, select Custom: Example Bracket Format as the source and Kakuyomu as the target.
6. Convert author-work-sample-custom-format.txt and compare it with author-work-sample-roundtrip-kakuyomu.txt and author-work-sample-kakuyomu.txt.

CLI validation
py -3 src\cli\rubimorph.py --validate-profile examples\custom-profiles\example-bracket-format.rubimorph-profile.json

CLI conversion checks
py -3 src\cli\rubimorph.py --from kakuyomu --to-profile examples\custom-profiles\example-bracket-format.rubimorph-profile.json --input examples\custom-profiles\example-input.txt --output examples\custom-profiles\example-expected-output.txt
py -3 src\cli\rubimorph.py --from kakuyomu --to-profile examples\custom-profiles\example-bracket-format.rubimorph-profile.json --input examples\custom-profiles\author-work-sample-kakuyomu.txt --output examples\custom-profiles\author-work-sample-custom-format.txt
py -3 src\cli\rubimorph.py --from-profile examples\custom-profiles\example-bracket-format.rubimorph-profile.json --to kakuyomu --input examples\custom-profiles\author-work-sample-custom-format.txt --output examples\custom-profiles\author-work-sample-roundtrip-kakuyomu.txt
