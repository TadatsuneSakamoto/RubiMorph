概要

目次へ戻る

RubiMorphは、Web小説投稿サイトや文書形式ごとに異なるルビ・傍点・注記の記法を変換するWindows用ソフトです。

GUIとCLIの両方に対応します。変換処理はローカルPC上で行い、原稿本文を外部サーバーへ送信しません。

インストール、Portable版、最初の変換だけを短く確認したい場合は 使用ガイド を参照してください。

ファイル変換単一ファイル、複数ファイル、フォルダ一括を選んで変換します。

ルビ出力

目次へ戻る

「ルビ出力」は、変換先がプレーンテキストの場合だけ有効です。プレーンテキスト以外では「対象外」と表示され、操作できません。

変換先表示変換時の扱い
プレーンテキスト有効「ルビを削除」または「親文字（ルビ）で残す」を使用します。
HTML、カクヨム、なろう、pixiv小説、青空文庫、Markdown対象外各形式のrendererが固定の記法で出力します。
カスタム形式対象外プロファイルの出力テンプレートがルビ記法を決定します。

プレーンテキストへ戻すと、前回選んでいた有効値が復元されます。テキスト変換とファイル変換は同じ判定を使います。

ステータス、ログ、警告

目次へ戻る

メイン画面下部には、直前の操作結果が短く表示されます。

状態例
成功変換が完了しました。 / 3ファイルを変換しました。 / 結果をクリップボードへコピーしました。
警告あり変換が完了しました。警告が2件あります。重要な警告の短い概要
失敗変換できませんでした。詳細を確認してください。

詳細ボタンを開くと、変換ログ、警告全文、対象ファイル、内部診断を選択・コピーできます。メイン画面は短い概要、詳細ウィンドウは全文という役割分担です。

新しい操作を実行すると、メイン画面の表示はその操作結果へ更新されます。過去の警告は現在の警告件数としては残りません。

テキスト変換

目次へ戻る

  1. 変換元と変換先を選びます。
  2. 入力欄に本文を貼り付けます。
  3. 必要に応じて「ルビ出力」を選びます。
  4. 「変換」を押します。
  5. 結果欄とステータスを確認します。
  6. 必要なら「コピー」で結果をクリップボードへ送ります。

変換元と変換先のどちらかにカスタム形式を使う場合は、先にカスタム形式管理画面でプロファイルを登録・有効化してください。

ファイル変換

目次へ戻る

単一ファイル

入力ファイルと出力先を選び、1ファイルだけを変換します。入力と同じパスを出力先にした場合は、上書きを避けるため変換済みファイル名へ退避します。

複数ファイル

選択した.txtまたは.mdを、指定した出力フォルダへ変換します。対象外拡張子は変換対象に含めません。

フォルダ一括

入力フォルダ配下の.txtと.mdを再帰的に探し、サブフォルダ構造を保って出力します。

カスタム形式

目次へ戻る

カスタム形式プロファイルは、独自記法を入力形式または出力形式として追加するUTF-8 JSONファイルです。任意のPython、JavaScript、シェル、外部プログラムは実行しません。

操作役割
新規作成プロファイルを作成して保存します。
編集・複製既存プロファイルを変更またはコピーします。
検証JSON構造、正規表現、テンプレート、上限値を確認します。
有効化・無効化変換元・変換先一覧に表示するかを切り替えます。
インポート・エクスポート外部ファイルと登録済みプロファイルをやり取りします。
削除登録済みプロファイルを削除します。

仕様の詳細は配布物内のdocs/custom-format-profiles.mdも参照できます。

カスタム形式サンプルを試す

目次へ戻る

サンプル一式をダウンロード

ファイル役割
example-bracket-format.rubimorph-profile.jsonサンプルプロファイル
example-input.txt短い確認用入力
example-expected-output.txt確認用入力の期待結果
author-work-sample-kakuyomu.txtカクヨム形式の小説本文サンプル
author-work-sample-custom-format.txtRubiMorphで生成したカスタム形式結果
author-work-sample-roundtrip-kakuyomu.txt逆変換で生成したカクヨム形式結果
  1. サンプル一式をダウンロードして展開します。
  2. RubiMorphで「カスタム形式」メニューを開きます。
  3. カスタム形式管理画面を開きます。
  4. example-bracket-format.rubimorph-profile.jsonをインポートします。
  5. 「検証」を実行します。
  6. 登録済みプロファイルを有効化します。
  7. 変換元に「カクヨム」を選びます。
  8. 変換先に「カスタム: Example Bracket Format」を選びます。
  9. author-work-sample-kakuyomu.txtを変換します。
  10. author-work-sample-custom-format.txtと比較します。
  11. 必要に応じて、変換元を「カスタム: Example Bracket Format」、変換先を「カクヨム」にして逆変換します。
  12. author-work-sample-roundtrip-kakuyomu.txtおよび原文と一致することを確認します。

サンプル本文のクレジット: 著者: 坂本忠恆 / 用途: RubiMorphの記法変換サンプル / 坂本忠恆による小説本文の抜粋

CLI

目次へ戻る

RubiMorph.exe --version
RubiMorph.exe --list-platforms
RubiMorph.exe --matrix
RubiMorph.exe --validate-profile profile.rubimorph-profile.json
RubiMorph.exe --from kakuyomu --to html --text "本文"
RubiMorph.exe --from kakuyomu --to plain --plain-ruby-mode remove --input input.txt --output output.txt
RubiMorph.exe --from kakuyomu --to plain --plain-ruby-mode parentheses --input input.txt --output output.txt
RubiMorph.exe --from kakuyomu --to-profile profile.rubimorph-profile.json --input input.txt --output output.txt
RubiMorph.exe --from-profile profile.rubimorph-profile.json --to kakuyomu --input input.txt --output output.txt
RubiMorph.exe --from kakuyomu --to html --input-dir input_folder --output-dir output_folder
RubiMorph.exe --from kakuyomu --to html --input input.txt --output output.txt --dry-run
RubiMorph.exe --from kakuyomu --to html --input input.txt --output output.txt --report report.json

--fromと--from-profile、--toと--to-profileは同時に指定できません。--plain-ruby-modeはプレーンテキスト出力時のルビ扱いです。

保存場所とバックアップ

目次へ戻る

登録済みカスタム形式プロファイルは通常、次の場所に保存されます。

%APPDATA%\RubiMorph\profiles

重要なプロファイルは、管理画面のエクスポート機能またはフォルダのコピーでバックアップしてください。

制限事項

目次へ戻る

  • 投稿サイト側の仕様変更により、変換結果が期待と異なる場合があります。
  • 情報量の少ない形式へ変換すると、傍点、注記、HTML構造などが失われる場合があります。
  • カスタム形式プロファイルにはサイズ、規則数、正規表現長、テンプレート長の上限があります。
  • 変換後の原稿は投稿前に必ず目視確認してください。

エラーとトラブル対応

目次へ戻る

状況確認すること
起動できないPortable版で_internalを削除していないか確認します。
変換できないステータスと「詳細...」のログを確認します。
警告が出る警告件数と詳細内容を確認し、投稿前に変換結果を目視確認します。
カスタム形式が一覧に出ないプロファイルが検証済み、有効、該当capabilityありか確認します。
Webページが開けないブラウザまたはネットワーク設定を確認し、表示されたURLを手動で開きます。

更新とアンインストール

目次へ戻る

最新版は GitHub Releases で確認できます。インストーラー版は新しいインストーラーを実行して更新します。

Portable版は新しいZIPを別フォルダへ展開し、必要なプロファイルや作業ファイルを移してください。

アンインストール前に、必要なカスタム形式プロファイルをバックアップしてください。
